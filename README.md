# CSU-bot — Viking Chat Assistant

A responsive, dark-themed chat interface built from scratch with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean, hand-written code.

## Features

- Modern dark UI with smooth message animations
- Animated typing indicator with variable response delay
- Keyword-based reply engine with natural fallbacks
- Auto-resizing input textarea
- Timestamped messages
- Mobile responsive layout
- Clear chat functionality

## Project Structure

```
CSU-bot/
├── index.html   # Markup and layout
├── style.css    # All styles, themes, and animations
├── app.js       # Bot logic, event handling, DOM updates
└── README.md    # Project docs
```

## Getting Started

Just open `index.html` in any browser — no build step, no installs needed.

```bash
# Clone the repo
git clone https://github.com/priyakrishna9919/CSU-bot.git

# Open in browser
open index.html
```

## Customization

- **Bot name**: edit `BOT_NAME` in `app.js`
- **Reply logic**: add entries to `replyMap` in `app.js`
- **Colors/theme**: update CSS variables at the top of `style.css`

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid/flexbox)
- Vanilla JavaScript (ES6+)
